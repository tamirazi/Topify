Topify

Topify is a progressive web application built with Angular & Spotify’s web api. designed with a clean UI & an intuitive UX, the application displays the user’s top artists, tracks, albums & genres and suggests playlists based on them.

Spotify access

Topify requires a Spotify account. It also needs access to your Spotify account.

Privacy

Topify is open source and works as client side. Topify does not save your Spotify data to any server.
